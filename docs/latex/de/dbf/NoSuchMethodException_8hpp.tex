\doxysubsection{Detailed Description}
A No\+Such\+Method\+Exception is thrown whenever a method which had been attempted to be get via reflexpr TS or any other kind of reflection feature that will be implemented later in C++, has not been found.

Since reflexpr TS is experimental this exception could seem as useless, indeed it is though for a later feature.
