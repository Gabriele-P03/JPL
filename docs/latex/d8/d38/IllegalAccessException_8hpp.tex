\doxysubsection{Detailed Description}
An Illegal\+Access\+Exception is thrown whenever a non-\/public field is attempted to be get via reflexpr TS or any other kind of reflection feature that will be implemented later in C++.

Since reflexpr TS is experimental this exception could seem as useless, indeed it is though for a later feature.
